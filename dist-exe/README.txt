# Maulwurf G코드 에디터 배포 파일

이 폴더에는 Maulwurf G코드 에디터의 다양한 배포 파일이 포함되어 있습니다.

## 파일 설명

1. `Maulwurf-v0.1.1.exe` - 독립 실행형 실행 파일 (최신 버전)
   - 설치 없이 바로 실행할 수 있는 파일입니다.
   - 일부 시스템에서는 필요한 종속성이 없을 수 있으므로, 실행에 문제가 있다면 아래의 설치 프로그램을 사용하세요.

2. `Maulwurf-v0.1.1-Setup.exe` - NSIS 설치 프로그램 (최신 버전)
   - 일반적인 Windows 설치 프로그램입니다.
   - 필요한 모든 종속성을 설치합니다.
   - 시작 메뉴에 바로가기를 생성합니다.

3. `Maulwurf.exe` / `Maulwurf_0.1.0_x64-setup.exe` / `Maulwurf-Setup.exe` / `Maulwurf-Installer.msi` - 이전 버전 파일

## 시스템 요구 사항

- Windows 10 이상
- 64비트 운영 체제

## 설치 방법

### 독립 실행형 실행 파일 사용
1. `Maulwurf-v0.1.1.exe` 파일을 원하는 위치에 복사합니다.
2. 파일을 더블 클릭하여 실행합니다.

### 설치 프로그램 사용
1. `Maulwurf-v0.1.1-Setup.exe` 파일을 더블 클릭합니다.
2. 설치 마법사의 지시에 따라 설치를 완료합니다.

## 새로운 기능

### v0.1.1 업데이트
1. **폴더 접근 권한 개선**
   - 폴더 접근 권한 오류 발생 시 더 자세한 오류 메시지 제공
   - 관리자 권한으로 다시 시작하는 옵션 추가

2. **디버그 콘솔 추가**
   - F12 키를 연속으로 3번 누르면 디버그 콘솔이 나타납니다.
   - 애플리케이션에서 발생하는 모든 로그와 오류를 확인할 수 있습니다.
   - 문제 해결 및 디버깅에 유용합니다.

## 문제 해결

- 독립 실행형 EXE 파일 실행 시 오류가 발생하면 설치 프로그램을 사용하여 필요한 종속성을 설치하세요.
- 폴더 접근 권한 오류가 발생하면 관리자 권한으로 실행하세요.
- 디버그 콘솔(F12 키 3번)을 통해 자세한 오류 정보를 확인할 수 있습니다.
